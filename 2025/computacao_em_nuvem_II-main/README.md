# 📌 Primeiro contato com PHP (Formulário simples)

## 📖 Sobre
Este projeto foi desenvolvido no início da minha jornada com programação, com o objetivo de entender o funcionamento básico do PHP e a interação com formulários HTML.

## 🧠 O que aprendi
- Execução de código PHP no servidor
- Criação de formulários com HTML
- Envio de dados com método POST
- Captura de dados usando $_POST

## ⚠️ Observações
O projeto possui uma estrutura simples e não segue boas práticas atuais, pois foi desenvolvido durante meus primeiros estudos.

## 🚀 Melhorias futuras
- Separação correta entre arquivos (HTML e PHP)
- Validação de dados do formulário
- Melhor organização do código
