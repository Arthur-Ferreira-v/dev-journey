# 📱 Quiz Android

## 📖 Sobre
Projeto desenvolvido com o objetivo de praticar desenvolvimento Android nativo utilizando Java e XML.

A aplicação simula um sistema de perguntas e respostas (quiz), utilizando componentes básicos de interface e lógica de controle.

---

## 🧠 O que aprendi
- Estrutura básica de projetos Android
- Criação de interfaces com XML
- Uso de componentes como:
  - TextView
  - Button
  - RadioGroup
  - RadioButton
- Manipulação de arrays em Java
- Controle de perguntas e respostas

---

## ⚙️ Tecnologias utilizadas
- Java
- Android Studio
- XML

---

## 🚧 Status do projeto
Projeto em desenvolvimento e aprendizado.

Algumas funcionalidades ainda não foram finalizadas, como:
