# 📌 Aplicação PHP com Docker

## 📖 Sobre
Este projeto foi desenvolvido com o objetivo de compreender a execução de aplicações PHP em um ambiente isolado utilizando Docker.

- Execução de aplicações PHP em servidor local
- Criação de ambiente com Docker
- Conceito de containerização
- Estrutura básica de um Dockerfile

## ⚙️ Tecnologias utilizadas
- PHP
- Docker
