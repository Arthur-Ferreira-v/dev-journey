# 📌 Formulário Web com PHP, MySQL e Docker

## 📖 Sobre
Projeto desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web utilizando PHP, HTML/CSS, conexão com banco de dados MySQL e containerização com Docker.

A aplicação utiliza formulários simples e conexão com banco de dados para fins de aprendizado.

---

## 🧠 O que aprendi
- Criação de formulários HTML
- Estilização com CSS
- Manipulação de dados com PHP
- Uso de parâmetros GET
- Sanitização básica com `htmlspecialchars`
- Conexão com banco de dados utilizando PDO
- Configuração de ambiente com Docker

---

## ⚙️ Tecnologias utilizadas
- PHP
- HTML5
- CSS3
- MySQL
- Docker
- Apache
